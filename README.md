# ConstellationCommand

**A gamified sci-fi mission control interface for Claude Code.**

Turn your software development workflow into a starship command experience. Projects are planets. AI agents are orbiting moons. Your terminal is a bridge console.

## Quick Start
```bash
# Clone
git clone https://github.com/YOUR_USERNAME/ConstellationCommand.git
cd ConstellationCommand

# Install
npm install

# Dev mode
npm run dev

# Open in browser
open http://localhost:5173
```

## Features (In Development)
- Tactical viewscreen with animated planet/moon visualization
- Real-time Claude Code agent monitoring
- Embedded terminal consoles per agent
- Mission planning and task management
- Project incubator galaxy map
- System logs and activity feeds

## Architecture
Node.js + Express + WebSocket backend -> React + TypeScript frontend. Designed to run on a headless server, accessed via browser.
